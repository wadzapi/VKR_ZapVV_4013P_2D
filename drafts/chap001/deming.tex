\subsubsection{Методологическая основа улучшения процессов}
В качестве основной методологической основы процесса улучшения бизнес-процессов можно выделить концепцию цикла Шухарта-Деминга, описанную в работе Деминга \cite[с. 95]{Deming_2007} со ссылкой о заимствовании у Шухарта. Цикл Шухарта-Деминга представляет собой модель жизненного цикла деятельности по улучшению процессов предприятия (см. диаграмму в Приложении~\ref{DemingCycle} на стр.~\pageref{DemingCycle}), составленную из набора циклически повторяемых групп деятельностей следующего вида:
\begin{description}
  \item[Планирование изменений] ---\, деятельности, направленные на сбор и анализ сведений о текущем состоянии процессов; планирование испытаний; выработку набора мер изменений, направленных на совершенствование процессов.
  \item[Осуществление спланированных изменений] ---\, деятельности по внедрению запланированных изменений, интеграцию новых решений с бизнес-средой, информирование о новых методах организации деятельности. 
  \item[Наблюдение эффекта изменений] ---\, деятельности по выявлению эффектов, связанных с внесенными изменениями; деятельности, связанные с количественной оценкой эффекта изменений.
  \item[Изучение результатов] ---\, деятельности по качественной оценке полученных результатов; выработка мер, направленных на выявления причинно-следственной связи; выработка подходов к проведению следущих итераций улучшения процессов.
\end{description} 
Следуя английским названиям составлящих модель категорий деятельности цикл Деминга-Шухарта в литературе (см. например: \cite[с. 4.8]{Juran_1999}) также называют циклом PDCA (Plan -- Do -- Check -- Action).
