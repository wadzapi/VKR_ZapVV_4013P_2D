С точки зрения функционального подхода в управлении, закрепленного в работе Файоля \cite{Fayol_1923}, вся совокупность операций, составляющих деятельность предприятия, может быть предаставлена разделением по функциональному признаку в виде множества гпупп. Модель ирархической структруры управляемой организации получают сопоставлением каждой из групп приоритета, задающего уровень иерархии функциональной деятельности. Полученную статическую организацию Файоль дополняет методологией, предназначенной для обеспечения и поддержания эффективности деятельностей, определяющей принципы разделения труда, обеспечение централизованного уравления с передачей управляющей информации по цепочке иерархии. Понятие иерархической модели функционального управлении уточняется в виде пирамидальной модели, выделением тенденции к повышению требований к квалификации, повышению уровня полномочий и снижению количества сотрудников при рассмотрении верхних слоев модели иерархии.
