Одной из неоспоримых тенденций в развитии подходов к разработке и проектированию является применение набора шаблонных методов и коцепций дизайна. В работах Барроса \cite{BarrosJulio_2011}, \cite{Barros_2007} разработаны концепции шаблона бизнес-процесса и типовой структуры бизнес-объектов. 
{\em Шаблон бизнес-процесса} представляет собой обобщенную модель процесса, описывающую поведение на основе лучших практик из области деятельности. В качестве примеров для наборов шаблонных бизнес-процессов можно привести \textemdash eTOM \cite{eTOM} для телекоммуникации и SCOR \cite{SCOR_11} для организации цепи поставок. {\em Типовая структура бизнес-объектов} предназначена для предоставления шаблона классификации процессов по классам, состав которых зависит от конкретной области. Наиболее известен каркас классификации, предлагаемый компанией APQC \cite{APQC}.
