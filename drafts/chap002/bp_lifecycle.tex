
В качестве модельного представления бизнес-процессов применяют концепцию цикла Шухарта-Деминга, описанную в работе Деминга \cite[с. 95]{Deming_2007} со ссылкой о заимствовании у Шухарта. Цикл Шухарта-Деминга представляет собой модель жизненного цикла деятельности по улучшению процессов предприятия (см. диаграмму в Приложении~\ref{DemingCycle} на стр.~\pageref{DemingCycle}), составленную из набора циклически повторяемых групп деятельностей следующего вида:
\begin{description}[itemsep=0pt, parsep=0pt]
  \item[Планирование изменений] \textemdash\, деятельности, направленные на сбор и анализ сведений о текущем состоянии процессов; планирование испытаний; выработку набора мер изменений, направленных на совершенствование процессов.
  \item[Осуществление спланированных изменений] \textemdash\, деятельности по внедрению запланированных изменений, интеграцию новых решений с бизнес-средой, информирование о новых методах организации деятельности. 
  \item[Наблюдение эффекта изменений] \textemdash\, деятельности по выявлению эффектов, связанных с внесенными изменениями; деятельности, связанные с количественной оценкой эффекта изменений.
  \item[Изучение результатов] \textemdash\, деятельности по качественной оценке полученных результатов; выработка мер, направленных на выявления причинно-следственной связи; выработка подходов к проведению следущих итераций улучшения процессов.
\end{description}
