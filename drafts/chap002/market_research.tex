\paragraph{Составление исследуемой совокупности СУБП.} \label{market_research}
  Для полноценного изучения функциональных особенностей современных СУБП необходимо сформировать выборочную совокупность, отражающую не только аспекты промышленных решений, но и применение в условиях малого бизнеса, некоммерческих целях. В целях уменьшения смещенности выборки, составленной на предварительном этапе (см.~\ref{market_trends} на стр. \pageref{market_trends}), добавим в рассмотрение системы управления с открытой или бесплатной лицензией, системы управления научной деятельностью.

В итоге для рассмотрения предлагается следующий набор программных СУБП:
\begin{description}[itemsep=0pt, parsep=0pt]
  \item[коммерческие системы управления] : Appian BPM  Suite\cite{AppianURL}, OpenText MBPM \cite{OpenTextMBPMURL}, Cordys BPMS \cite{CordysURL}, Pegasystems BPM \cite{PegaSystemsURL}, Fujitsu Interstage BPM \cite{FujitsuInterstageURL}, Tibco Active Matrix BPM \cite{TibcoURL};
  \item[некоммерческие системы управления]: jBPM \cite{jBPMURL}, BonitaBPM \cite{BonitaURL}, Taverna Workflow \cite{TavernaWorkflowURL}, Activiti BPM \cite{ActivitiURL}, Pegasus \cite{PegasusURL}, Intalio BPMS \cite{IntalioURL}.
\end{description}

\paragraph{Критерии сравнения СУБП.}
Сравнительный анализ современных систем управления проведен на основе выявленных в ходе теоретического исследования групп функциональных критериев. На первом этапе анализа проведена оценка функциональных возможностей СУБП:
\begin{itemize}[itemsep=0pt, parsep=0pt]
  \item функции моделироания и проектирования бизнес-процессов; 
  \item функции организации структуры выполнения работ и документооборота;
  \item функции динамического контроля выполнения работ;
  \item функции распределения и контроля ресурсов.
\end{itemize}

Затем проведена  оценка показателей для  функций, характерных системам управления третьего поколения: 
\begin{itemize}[itemsep=0pt, parsep=0pt]
  \item функции организации виртуального предприятия, распределенной работы;
  \item функции управления жизненным циклом процессов;
  \item функции управления и разделения знаний;
  \item функции обеспечения гибкости бизнеса и управления.
\end{itemize}

Согласно выделенным критериям, для каждой из рассматриваемых СУБП было проведено оценивание характеристик по 10-ти бальной шкале, результаты которого сведены в таблицы (см. Приложение~\ref{bpms_research_table}). Полученные данные позволяют оценить функциональные требования отрасли к СУБП, установить связь с теоретическими исследованиями, сформировать представление о реализации концепций новой парадигмы в управленческой деятельности.
