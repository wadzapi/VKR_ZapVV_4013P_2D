\paragraph{Составление исследуемой совокупности СУБП.}
  Для полноценного изучения функциональных и архитектрурных особенностей современных СУБП необходимо сформировать выборочную совокупность, отражающую не только аспекты промышленных решений, но и применимых в условиях малого бизнеса, некоммерческих целях. В целях уменьшения смещенности выборки, составленной на предварительном этапе (см.~\ref{market_trends} на стр. \pageref{market_trends}), добавим в рассмотрение системы управления с бесплатной лицензией, системы управления научной деятельностью.

В итоге для рассмотрения предлагается следующий набор программных СУБП:
\begin{description}[itemsep=0pt, parsep=0pt]
  \item[коммерческие системы управления] : Appian BPM  Suite\cite{AppianURL}, OpenText MBPM \cite{OpenTextMBPMURL}, Cordys BPMS \cite{CordysURL}, Pegasystems BPM \cite{PegaSystemsURL}, Fujitsu Interstage BPM \cite{FujitsuInterstageURL}, Tibco Active Matrix BPM \cite{TibcoURL};
  \item[некоммерческие системы управления]: jBPM \cite{jBPMURL}, BonitaBPM \cite{BonitaURL}, Taverna Workflow \cite{TavernaWorkflowURL}, Activiti BPM \cite{ActivitiURL}, Pegasus \cite{PegasusURL}, Intalio BPMS \cite{IntalioURL}.

\paragraph{Критерии сравнения СУБП.}
Сравнительный анализ современных систем управления проведен на основе выявленных в ходе теоретического исследования групп функциональных критериев. На первом этапе анализа  были выделены функциональные возможности СУБП:
\begin{itemize}[itemsep=0pt, parsep=0pt]
  \item функции моделироания и проектирования бизнес-процессов ; 
  \item функции организации структуры выполнения работ и документооборота;
  \item функции динамического контроля выполнения работ;
  \item функции распределения и контроля ресурсов.
\end{itemize}

Затем проведено сравнение на основе выделения функций, характерных для систем управления третьего поколения: 
\begin{itemize}[itemsep=0pt, parsep=0pt]
  \item функции организация виртуального предприятия, распределенной работы;
  \item функции управления жизненным циклом процессов;
  \item функции управления и разделения знаний;
  \item функции обеспечения гибкости бизнеса и управления.
\end{itemize}

Согласно выделенным критериям, для каждой из рассматриваемых систем было проведено оценивание характеристик по 10-ти бальной шкале (см.~\ref{bpms_research_table}). Полученные данные позволяют оценить функциональные требования отрасли к СУБП, установить связь с теоретическими исследованиями, сформировать представление о реализации концепций новой парадигмы в управленческой деятельности.   

\paragraph{Заключение}
Современные системы в полной мере оснащены функциями, присущими традиционным системам управления. Таким образом, наличие функционала традиционного управления не является конкурентным преимуществом, а скорее отражает набор функций, которые стали привычными для управленца. На рынке программ СУБП наблюдается тендеция к переходу на новую парадигму управления. Именно функционал систем интеллектуального и гибкого направления может рассматриваться в качестве направления, перспективного для дальнейшего исследования и разработки.
